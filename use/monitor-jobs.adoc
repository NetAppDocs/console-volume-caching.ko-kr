= 감사 페이지에서 볼륨 캐싱 작업 모니터링
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp 콘솔 감사 페이지를 사용하여 모든 볼륨 캐싱 작업을 모니터링하고 진행 상황을 확인할 수 있습니다.

.단계
. NetApp 콘솔 메뉴에서 *관리* > *감사*를 선택합니다.
. 선택적으로 시간, 서비스, 작업, 에이전트, 리소스, 사용자 또는 상태별로 필터링할 수 있습니다.
. 모든 캐시와 작업을 탐색해 보세요.
