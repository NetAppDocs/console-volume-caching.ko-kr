= 볼륨 캐싱 관련 자주 묻는 질문
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
이 FAQ는 질문에 대한 빠른 답변을 찾는 데 도움이 될 수 있습니다.

볼륨 캐싱 URL은 무엇인가요? 브라우저에서 URL을 확인하려면 다음을 입력하세요. https://console.netapp.com/["https://console.netapp.com/"^] BlueXP 콘솔에 접속하려면.

볼륨 캐싱을 사용하려면 라이선스가 필요합니까? NetApp 라이선스 파일(NLF)은 필요하지 않습니다.

볼륨 캐싱을 활성화하는 방법은 무엇인가요? 볼륨 캐싱은 별도의 활성화 설정이 필요하지 않습니다. 볼륨 캐싱 옵션은 NetApp Console 왼쪽 탐색 메뉴에 자동으로 나타납니다.
